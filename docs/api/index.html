HOW TO INSTALL

To install with RubyGems simply open a console and type: 

  gem install cuts

To manually installation, you will need the Ruby Setup package (setup.rubyforge.org). Then download the tgz package and type: 

  $ tar -xvzf cuts-x.y.z.tgz
  $ cd cuts-x.y.z.tgz
  $ sudo setup.rb
